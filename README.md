# Cemig Bot (Telegram + WhatsApp Automation)

## Setup

1. Instale dependências:
   pip install -r requirements.txt

2. Instale browsers do Playwright:
   playwright install

3. Configure o .env com seu BOT_TOKEN

4. Rode o projeto:
   python main.py

## Primeiro uso

- Vai abrir o WhatsApp Web
- Escaneie o QR Code
- A sessão será salva automaticamente

## Uso

- Envie qualquer mensagem no Telegram
- Digite CPF
- Digite número da instalação
- O bot retorna a resposta da Cemig
